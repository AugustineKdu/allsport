## 🔧 클라우드타입 배포 단계

### 1. GitHub 저장소 푸시
```bash
git add .
git commit -m "Ready for CloudType deployment"
git push origin main
```

### 2. 클라우드타입 설정
1. **새 프로젝트 생성**
2. **GitHub 연결** - 이 저장소 선택
3. **배포 설정**:
   - **런타임**: Docker
   - **포트**: 80
   - **자동 배포**: 활성화

### 3. 환경변수 (선택사항)
기본 설정으로 바로 작동하지만, 필요시 추가:
```
APP_URL=https://your-domain.cloudtype.app
APP_NAME=AllSports
```
